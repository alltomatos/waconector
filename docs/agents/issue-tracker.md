# Issue tracker

- **Sistema**: GitHub Issues, no próprio repositório (`alltomatos/waconector`).
- **Autenticação**: `gh` CLI já autenticado localmente (`gh auth status`); GitHub MCP também
  disponível para operações estruturadas (buscar, criar, comentar).
- **Templates**: `.github/ISSUE_TEMPLATE/` — `bug_report.yml`, `feature_request.yml`,
  `new_adapter.yml` (específico deste projeto, para pedidos de novo provider), `config.yml`
  (aponta para a metodologia de dossiê e para `CONTRIBUTING.md`).
- **PRs**: `.github/pull_request_template.md` — checklist de QA + checklist específico para PRs
  de adapter.
- Não há tracker externo (Jira/Linear/GitLab) — GitHub Issues é a única fonte de verdade.
